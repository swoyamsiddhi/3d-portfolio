## 🚀 About The Project

This repository contains the source code for my personal portfolio. Moving away from traditional static designs, this site leverages WebGL and advanced scroll-trigger animations to create a cinematic experience. The centerpiece is a 3D astronaut model—as you scroll, the camera dynamically zooms right into the astronaut's helmet, seamlessly transitioning into my project showcases and technical background.

---

## ✨ Key Features

* **🎬 Cinematic Scroll Animation:** Deep integration with GSAP's ScrollTrigger to map the user's scroll position directly to the 3D camera coordinates.
* **🧑‍🚀 3D Rendering:** High-fidelity 3D astronaut model loaded and manipulated using Three.js.
* **⚡ High Performance:** Optimized render loops and asset loading to ensure a buttery smooth 60fps experience.
* **📱 Responsive Architecture:** The 3D canvas scales and adapts intelligently across desktop, tablet, and mobile breakpoints.

## 💻 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have a modern web browser and a local development server (like the VS Code Live Server extension) installed to prevent CORS issues when loading 3D `.glb` or `.gltf` models.

### Installation

1.  **Clone the repo**
    ```sh
    git clone [https://github.com/your_username/portfolio-3d.git](https://github.com/your_username/portfolio-3d.git)
    ```
2.  **Navigate to the directory**
    ```sh
    cd portfolio-3d
    ```
3.  **Launch the application**
    * Open the folder in VS Code.
    * Right-click `index.html` and select **"Open with Live Server"**.
